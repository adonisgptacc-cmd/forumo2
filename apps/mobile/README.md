# Forumo Mobile (React Native)

The mobile client will be built with Expo and leverage the same API + component tokens as the web app.

## Planned MVP features

- Auth + onboarding with KYC reminders
- Listing discovery + filters
- Messaging inbox (read + reply)
- Escrow order tracking
- Push notifications via Expo services

## Setup (coming soon)

pnpm create expo-app apps/mobile --template
